Readme File

Program 1 - Preprocessing.py
- This program preprocesses the tweets and gives a file called Preprocessed.csv.

Program 2 - Sentimental Analysis.py
- This program uses the Preprocessed file to generate two files. Sentiment_Data is used for regression whereas the Location_Data gives us a clear score for various disorders according to each country.

Program 3 - Linear Regression.py
- This program uses the score generated from the Sentiment data to perform regression. The output is a series of plots generated.
